# deprem-guvenlik-sistemi
Arduino tabanlı otomatik deprem güvenlik sistemi.
MPU6050 sensörü ile sarsıntı algılama, buzzer ile alarm, röle ile güvenli kapatma.
## Donanım
- Arduino Uno
- MPU6050 sensör
- Buzzer
- Röle modülü
- LCD (opsiyonel)
